# install
```bash
git clone http://github.com/nimacpp/ip-info.git
cd ip-info
ruby ip.rb
```

# USE IP-info

```bash
chmod +x ip.rb
./ip
```

### or 

```bash 
chmod +x ip.rb
./ip 127.0.0.1 # enter in ARGV target ip
```

You do not need an extra library
